# Recommender Systems

This project implements a recommender system in Google Colab, covering data preprocessing, similarity-based recommendation logic, and evaluation. The notebook demonstrates end-to-end workflow for building and analyzing recommendation outputs.

## TL;DR
A complete Colab-based recommender system implementation including data processing, model building, predictions, and evaluation. Easy to run in Colab or locally.

## Files Included
- `Recommender_Systems.ipynb` — main notebook
- `requirements.txt` — inferred Python dependencies
- `LICENSE` — MIT license
- `COMMIT_MESSAGE.txt` — suggested commit message for GitHub

## Run in Google Colab (Recommended)
1. Go to https://colab.research.google.com  
2. Click **File → Upload notebook**  
3. Upload `Recommender_Systems.ipynb` and run cells

## Run Locally
```bash
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
jupyter notebook Recommender_Systems.ipynb
